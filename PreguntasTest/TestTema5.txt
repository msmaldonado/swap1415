1. Qué paso no se corresponde con los pasos para establecer una Conexión HTTP
	a) El cliente envia un paquete ACK como respuesta.
	b) El cliente inicia conexión HTTP enviando un paquete TCP SYN al puerto 80 del servidor weeb
	c) El cliente envía un paquete ACK como entrada.
	d) El servidor web envía un paquete ACK al cliente seguido de otro SYN
	Respuesta correcta: c
